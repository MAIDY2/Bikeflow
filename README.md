Descripción del proyecto: 

BikeFlow es una aplicación web y móvil desarrollada para optimizar la gestión del préstamo, devolución y mantenimiento de bicicletas dentro de la institución.

Actualmente, estos procesos se realizan de manera manual, lo que genera problemas como pérdida de información, falta de control sobre las bicicletas disponibles y ausencia de un historial de préstamos.

La plataforma permitirá administrar el inventario de bicicletas, gestionar usuarios, registrar préstamos y devoluciones, actualizar el estado de las bicicletas y consultar el historial de uso, garantizando una administración más eficiente y segura.
